# Abu Usman Hospital and Health Services — Static Site

Pure HTML, CSS, and vanilla JavaScript. No build step, no framework.

## Run locally
Open `index.html` in any browser.

## Deploy to GitHub Pages
1. Push this folder to a GitHub repository (root).
2. In the repo: Settings → Pages → Deploy from branch → `main` / `/root`.
3. Done — the site loads instantly, no compilation needed.

## Structure
- `index.html`, `about.html`, `services.html`, `departments.html`, `doctors.html`, `pictures.html`, `testimonials.html`, `branches.html`, `contact.html`
- `style.css` — all styles
- `script.js` — mobile menu, testimonial carousel, doctor bio toggle, appointment form (mailto)
- `images/` — every image used on the site (replace by overwriting the file)

Icons: Font Awesome (CDN). Fonts: Google Fonts (Poppins).
